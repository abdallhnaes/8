# تطبيق زاجل - الحسابات والمزامنة والإشعارات

هذه النسخة تحافظ على جوهر التطبيق الداخلي والحسابات والتصميم، مع تحسين نظام الحماية والمزامنة.

## التعديلات الجديدة

- رمز المالك يطلب مرة واحدة فقط بعد تثبيت التطبيق على الجهاز.
- بعد قبول رمز المالك لا يطلب مرة ثانية على نفس الجهاز إلا إذا تم حذف بيانات التطبيق أو إعادة تثبيته.
- يوجد حساب مدير وحساب موظف.
- التمييز بين المدير والموظف يتم من خلال الحساب المختار ثم PIN الخاص بكل حساب.
- كل حساب يستطيع تغيير PIN الخاص به بنفسه من إعدادات التطبيق.
- أرقام PIN لا تظهر داخل التطبيق.
- رمز المالك لا يظهر داخل التطبيق ولا يمكن تعديله من الإعدادات.
- المدير يستطيع تعديل بيانات الموظف وإعادة تعيين PIN الموظف بكتابة رمز جديد فقط.
- الموظف لا يستطيع تعديل حسابه أو تعديل الفواتير أو حذفها.
- المزامنة عبر Firebase Realtime Database.
- أي عملية تعديل يقوم بها المدير ترسل إشعارًا للموظف.
- الرسائل بين الموظف والمدير ترسل إشعارًا على الهاتف.
- بقي النسخ الاحتياطي والاستيراد والطباعة كما هي.

## إشعارات الهاتف

أضفنا جسر Android للإشعارات، لذلك عند أول تشغيل قد يطلب الهاتف السماح بالإشعارات.

ملاحظة مهمة:
الإشعارات تعمل عندما يكون التطبيق مفتوحًا أو يعمل بالخلفية. للحصول على إشعارات مضمونة والتطبيق مغلق تمامًا، يلزم إضافة Firebase Cloud Messaging في نسخة لاحقة.
